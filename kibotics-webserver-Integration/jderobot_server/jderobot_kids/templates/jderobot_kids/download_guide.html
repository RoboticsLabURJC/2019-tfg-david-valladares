Instrucciones para la descarga del ejercicio

  1. Descarga tu ejercicio aquí: Descargar
  2. Conéctate a la red wifi de tu raspberry, con nombre de red: piBot(n)

    3. Para ordenadores con Sistema Operativo Linux o MacOSX

  3. Abre una consola y ejecuta: 
      scp ~/Descargas/ejercicio.zip pi@10.3.141.1:/home/pi/local_downloader/listening/

    4. Para ordenadores con sistema operativo Windows:

  4. Abrimos el programa WinSCP e introducimos los datos de nuestro PiBot
  5. Si aparece la siguiente ventana de notificación, pulsamos en Sí
  6. En la parte derecha aparece el contenido de la Raspberry pi. Entramos en la carpeta 
    /local_downloader/listener/
    y arrastramos el archivo desde la izquierda a la derecha.
  7. Por último, introducir la contraseña. Por defecto 
    raspberry
