Características


Kibotics es un entorno web para docencia en robótica y programación (STEM). Enseña de manera atractiva conceptos básicos de tecnología (informática, electrónica, ...) e inicia a niñ@s y adolescentes en robótica y programación de robots. Sigue un enfoque práctico, de aprender haciendo. Ayuda a estructurar el pensamiento, a organizar las acciones en pasos para resolver un problema y a formar espíritu analítico.

    Sin instalación.
    Windows, MacOS, Linux

  • Robots Reales y Simulados

    PiBot, drones, Fórmula 1

  • Lenguajes

    Python y Scratch

PiBot - Sigue Pelota

PiBot usando visión artificial:

Para este ejemplo, el PiBot es capaz de seguir la pelota naranja utilizando para ello la cámara que lleva incorporada y la visión artificial para localizarla y seguirla

Detalles:

  • Software: Python
  • Hardware: PiBot, Cámara, Motores
  • Ejercicio: PiBot siguiendo la pelota.

PiBot - Choca Gira

PiBot evitando obstáculos:

En este ejemplo el PiBot se detendrá, retocederá y girará cuando alcance un obstáculo en frente de su camino, corrigiendo la posición hacia una que no haya obstáculo.

Detalles:

  • Software: Python
  • Hardware: PiBot, Motores, Ultrasonidos
  • Ejercicio: PiBot Evitando obstáculos.

PiBot - Control de posición

PiBot usando el control de posición:

El PiBot tiene calibrados los motores de tal manera que es capaz de hacer giros perfectos y, por ejemplo, volver al punto de donde salió, con 4 giros de 90º

Detalles:

  • Software: Python.
  • Hardware: PiBot, Motores.
  • Ejercicio: PiBot usando el control de posición.
